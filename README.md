# yiff

yiff.cr is a port of [npm/yiff](https://npm.im/yiff) to Crystal

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     yiff:     
      # development is on:
      git: https://git.lio.cat/crystal/yiff.git
      # auto mirror:
      github: himbolion/yiff.cr
   ```

2. Run `shards install`
